Computes formula strings of the the real part of the spherical harmonics up to degree L (excluded). Variables are either spherical coordinates phi and theta (or cartesian coordinates x,y,z) on the UNIT SPHERE. Parameters

    Degree up to which to calculate the spherical harmonics (degree L is excluded).
